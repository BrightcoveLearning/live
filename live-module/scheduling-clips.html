In this topic, you will learn how to schedule a video clip using the Live module.

Scheduling a Clip in the Control Room

  1. In the Live module, create a new Live event with a Static Entry Point or visit a previously created one.
  2. In the Control Room, scroll down to the Scheduled Clips section.
  3. Click Add Scheduled Clip to schedule a new clip.
    Note: If the Scheduled Clips panel is not available in the Control Room, please contact Brightcove support.
