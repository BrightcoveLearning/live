Introduction

The Live scheduler allows you to schedule actions for Static Entry Point (SEP) jobs.. You just set up a scheduler workflow with the times, and the necessary API actions will be handled automatically. There are two kinds of scheduler workflows that you can set up, as detailed in the following sections.

Start/Stop scheduler

The scheduler allows you to set up activation and deactivation times for an SEP job, and you can also update the times. See details in Scheduling Activation/Deactivation of an SEP Stream.

Clip scheduler

You can also schedule the generation of a clip from your SEP job, and then update the schedule later or cancel the clip if something changes. See details in Scheduling Clip Generation for an SEP Stream.
